📝 Guide de l'éditeur HTML

L'identifiant, le type et le titre sont gérés automatiquement.

Utilisez l'éditeur visuel (Quill) pour formater votre contenu.

🎨 Formatage de base

Titres

Utilisez le menu déroulant de titres dans la barre d'outils :

  • H1 : Titre principal de page (avec trait vert dessous)
  • H2 : Sous-titre important
  • H3 : Section
  • Normal : Texte normal

⚠️ Note : "Huge" dans taille = juste texte plus grand. Pour de vrais titres avec style, utilisez H1/H2/H3 !

Mise en forme du texte

  • Gras : Sélectionnez le texte → clic sur B
  • Italique : Sélectionnez le texte → clic sur I
  • Souligné : Sélectionnez le texte → clic sur U
  • Couleur : Sélectionnez le texte → palette de couleurs
  • Surlignage : Sélectionnez le texte → surligneur

📐 Alignement

  • Sélectionnez le paragraphe → boutons d'alignement
  • Gauche, centre, droite, justifié
  • Pour l'arabe : direction RTL automatique

🖼️ Médias

Insérer une image (syntaxe spéciale) :

[IMAGE: fichier.jpg | 100% | Description facultative]
[IMAGE: fichier.jpg | 50% | Description plus petite]
[IMAGE: fichier.jpg | 25% | Miniature]

📌 Le pourcentage contrôle la taille (largeur/hauteur) de l'image

Insérer une vidéo :

[VIDEO: video.mp4 | 100% | Description facultative]
[VIDEO: video.mp4 | 50% | Vidéo plus petite]
[VIDEO: video.mp4 | 25% | Miniature vidéo]

📌 Le pourcentage contrôle aussi la taille des vidéos

📦 Blocs colorés

Pour créer un bloc avec couleur de fond :

  1. Sélectionnez votre texte
  2. Clic droit → "Inspecter l'élément"
  3. Ajoutez : style="background-color: #f0f8ff; padding: 1rem; border-radius: 5px;"

Ou utilisez l'outil de fond dans la barre d'outils.

📝 Variables dynamiques

Vous pouvez utiliser :

  • {{book.title}} – titre du livre
  • {{book.subtitle}} – sous-titre
  • {{book.institution}} – institution
  • {{book.years}} – années

💡 Conseils

  • Pour les titres de page : utilisez H1 et centrez-les
  • Pour faire ressortir du texte : gras + couleur
  • Pour les citations : italique + alignement centre
  • Testez les couleurs avec parcimonie
